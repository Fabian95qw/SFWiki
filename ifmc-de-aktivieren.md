# Beschreibung
Ein kleines Modul, welches es erlaubt per Modultaste IFMC für einen User zu De-/Aktivieren
# Konfiguration
Instanz erstellen, Benutzer für Taste auswählen.
Speichern und beim Benutzer auf den BLF-Platzieren.
Fertig!
