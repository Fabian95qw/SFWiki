# Beschreibung
Das Modul erlaubt das Kopieren von VIP Tasten zu Telefon Funktionstasten für Alle User, einzelne Gruppen, oder für User-Self Service via Anruf.

# Konfiguration

![1.png](/uploads/vop2functionkey/1.png)

## Beim Speichern / Übernehmen anwenden
Wendet die Einstellungen sofort an, statt Zeitgesteuert

## VIPs zu Funktionstasten übertragen für:
Definiert ob die VIP's für alle Benutzer oder nur für eine Gruppe von Benutzern auf die Funktionstasten übertragen werden soll.

## VIPs kopieren:
Definiert, ob nur eine gewisse Kategorie gemäss dessen Namen oder alle VIP's kopiert werden.

![2.png](/uploads/vop2functionkey/2.png)

## Kategoriename
Der Name der Kategorie, falls nicht alle Kategorien kopiert werden sollen

## Telefone provisionieren
Ob die Tischtelefone der User neu provisioniert werden sollen.

## Offset
Ein Basisoffset vom Start der Funktionstasten für alle Tasten
